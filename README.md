# AI Agent Skills

Centralized collection of reusable AI agent skills for data analytics, dashboards, documentation, and portfolio projects.

## Structure

```
skills/
├── analytics-insight-generation/
├── dash-dashboard-framework/
├── data-pipeline-automation/
├── data-quality-assessment/
├── docs-archive-organization/
├── ecommerce-analytics-workflow/
├── interview-prep-documentation/
├── mermaid-diagram-generation/
├── portfolio-dashboard-build/
└── star-schema-design/
```

## Usage

Skills follow the [Claude Agent Skills](https://platform.claude.com/docs/en/agents-and-tools/agent-skills/overview) format. Copy to your `.opencode/skills/` directory or configure your agent to reference this repo.
