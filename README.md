# Clozed Anki Generation

Create flashcards for spaced repetition learning, compatible with Anki import. Specializes in cloze deletion cards.

## Usage

This skill helps you create effective cloze deletion flashcards optimized for spaced repetition in Anki.

### Triggers

- Create flashcards
- Make Anki cards
- Spaced repetition
- Memory cards
- Cloze deletion cards
- Anki deck

## Features

- Cloze deletion rules and best practices
- Sentence framing patterns for clear, unambiguous cards
- Multi-cloze support for related paired facts
- Anki CSV export format
- Quality checklist for card validation
- Tagging strategy for filtered decks

## Format

Upload to Anki as CSV:

1. File → Import → select CSV
2. Note Type: **Cloze**
3. Field mapping: Column 1 → **Text**, Column 2 → **Extra**
4. Enable: **Allow HTML in fields**
